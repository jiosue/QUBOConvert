Polynomial Constrained Spin Optimization (PCSO)
===============================================

Accessed with ``qubovert.PCSO``. Note that it is important to use the ``PCSO.convert_solution`` function to convert solutions of the PUBO, QUBO, PUSO, or QUSO formulations of the PCSO back to a solution to the PCSO formulation.

We also discuss the ``qubovert.spin_var`` function here, which is just returns a ``PCSO``.
